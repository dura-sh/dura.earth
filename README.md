# The World of Dura.*

The first pit stop you need before entering the world of [Dura.*](https://github.com/durash/dura.world).

## Development

```bash
$ git clone git@github.com:durash/dura.world.git
$ cd dura.world
$ yarn
$ yarn start
```

Then visit http://localhost:8001 .

## Deploy

```
$ yarn site
```
